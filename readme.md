# Clever Domain Name Generator

Generates domains where the TLD forms part of a word.

	python tld.py

you must have '/usr/share/dict/words'

# Examples

fled.ge
angeli.na
supercla.im
chokebo.re
throwst.er
kneadi.ng
sabbati.ne
heteromorpho.is
debasing.ly
subun.it
metatheri.an
exhibitive.ly
panca.ke
